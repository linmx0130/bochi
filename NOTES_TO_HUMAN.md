Hi, human friends! This is a tool designed for AI agents to operate Android devices for debugging purposes. You may also use this tool if you think it's useful.

You may use following commands to install the [SKILL.md](SKILL.md) to your agents:
```bash
npx skills add https://github.com/linmx0130/bochi --skill operate-android-devices-with-bochi
```

Most code of this tool is authorized by [Kimi-CLI](https://github.com/MoonshotAI/kimi-cli/).
